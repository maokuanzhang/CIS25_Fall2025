### File Description
Module 9.cpp - contains the Item class, main function, and search function implementations.;<br>

### Program Description
This program proves the use of:<br>
1.Arrays – to store multiple objects of the Item class<br>
2.Loops – to input and display inventory items<br>
3.Functions – to search for an item by name and display its quantity<br>
4.Class usage – creating and accessing object members (name, quantity)<br>

### Program Function
The program performs basic inventory management tasks:<br>
1.Prompts the user to enter the name and quantity for five items<br>
2.Stores these items in an array of objects (Item items[5])<br>
3.Displays the entire inventory list using a loop<br>
4.Allows the user to search for an item by name and displays its quantity if found<br>
5.Supports multiple searches until the user types exit to quit<br>

This program creates a simple inventory management system using arrays, loops, and a class to store, display, and search items by name.
