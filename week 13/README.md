### Program Description
This program proves:<br>
1.Inheritance (base class → derived classes)<br>
2.Method overriding using the override keyword<br>
3.Runtime polymorphism using a virtual method<br>

### Program Function
The program represents three system roles:<br>
1.User: Basic system access<br>
2.Employee: Elevated access for modifying limited data<br>
3.InventoryManager: Full administrative access<br>

Finally, the program prints the correct access level message based on the user role.
